Course Description

“Design Fundamentals for the Web” is an introductory course allowing students to develop and practice specific, foundational skills in web production, while simultaneously developing an understanding of and strategies for effective visual communication in general. Students will learn when and how to use tools such as HTML, CSS and Javascript. Through readings, lectures and demonstrations, students will also develop an appreciation for fundamental issues of graphical communication, including color, typography and composition.

Students will learn to:

  • Structure a web site using hand-coded HTML
  • Stylize a web site using hand-coded CSS
  • Understand fundamental principles of graphic communications and apply them to the design of visually engaging web sites
  • Reconcile traditional approaches to design with the unique concerns of designing for a flexible and interactive medium that can be accessed through a wide variety of platforms
  • Explore current trends and emerging standards in interactive design
  • Organize content using standard tools and processes for information architecture
  • Plan and facilitate formal usability tests
